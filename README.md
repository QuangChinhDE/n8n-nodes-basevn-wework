# n8n-nodes-basevn-wework

This is an n8n community node for BaseVN - App Wework API integration.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### Resources

- **Project**: Create, edit, get full information, list projects
- **Task**: Create, create subtask, edit, edit extra, get, list by project, mark done, delete, add followers
- **Department**: Create, edit, get, list, remove
- **Tasklist**: Get tasklist

### Trigger Events

- On Task Created
- On Task Done
- On Task Status Updated
- On Subtask Created
- On Subtask Done
- On Subtask Status Updated

## Credentials

You need to configure the BaseVN Wework API credentials:
- **Domain**: Your BaseVN domain (e.g., `example.basevn.com`)
- **Access Token**: Your API access token

## Compatibility

Tested with n8n version 1.0+
